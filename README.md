# LaClass 3.0 Public

LaClass（ラクラス）は、先生向けの座席表・学級経営ツールです。

## 主な機能

- 座席表作成
- 貼り出し用・教卓用切り替え
- 出席番号順・縦
- 出席番号順・横
- 男女交互・縦
- 男女交互・横
- NGペア回避
- 周囲8方向NG
- 班分け
- 発表順
- 授業タイマー
- AI支援デモ
- Googleログイン導線
- ログイン時のみ保存できる導線
- AdSense広告枠
- SEO用コンテンツ

## 注意

現在のGoogleログインはデモ表示です。
本番でGoogleログインとクラウド保存を行うには、Firebase Authentication と Firestore の設定が必要です。

## GitHub Pages公開

このフォルダ内のファイルをGitHubリポジトリにアップロードし、Settings → Pages から公開してください。
